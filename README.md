# PostgreSQL SQL Learning Portfolio

This repository documents my structured SQL learning journey in PostgreSQL using the **dvdrental** sample database.

It is organized as a technical progression through foundational SQL topics. The repository currently contains topic-based learning scripts and will later expand into more advanced SQL practice and portfolio-style analysis projects.

## Objectives

- Build strong SQL fundamentals in PostgreSQL
- Practice writing clean, readable, and maintainable queries
- Develop analytical thinking through structured topic-based exercises
- Grow this repository into a stronger SQL portfolio over time

## Current Topics Covered

- SELECT
- WHERE
- ORDER BY
- GROUP BY
- INNER JOIN
- LEFT JOIN
- CASE WHEN
