# Rental Retention and Cohorts

I use this project to practice cohort analysis with SQL. The output tracks
customers by their first rental month and measures whether they rent again in
later months.

This is a sample-data cohort, so I keep the interpretation modest. The value is
in the SQL structure and validation.

## Main SQL patterns

- date truncation
- CTEs
- cohort month calculation
- distinct customer counts
- retention percentage
- validation of month zero cohort sizes
