# Findings

This project identifies category leaders by both revenue and rental count. Using
both metrics matters because the highest-rented film is not always the same as
the highest-revenue film.

## What I would look for in the output

- films that rank highly by both revenue and rental count
- categories where top films are close together
- categories where a few titles dominate
- whether rental rate affects revenue rank

## Why this matters in an interview

This project helps me discuss bridge tables, window ranking, top-N per group,
and revenue reconciliation. It also shows that I validate category assignment
before trusting category-level rankings.
